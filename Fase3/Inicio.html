Etiquetas Básicas de HTML, atributos y valores


  • Introduccion

    • El presente trabajo tiene como finalidad dar a conocer las importancia y uso que tienen las etiquetas HTML, pues es un lenguaje que nos permite crear paginas web y darles un aspecto visual para que el usuario tenga una mejor interaccion con el sitio. HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página web está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto. Estas etiquetas de marcado son las propias etiquetas HTML. Cuando escribes código en HTML, estás escribiendo etiquetas HTML.
